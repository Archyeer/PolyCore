# PolyCore

## 项目简介

PolyCore 是一个基于 Qt 框架的跨平台代理客户端，继承自 Qv2ray 项目。支持 Windows, Linux, macOS 等平台。

## 特性

- 跨平台支持
- 插件系统
- 现代化界面
- 高性能代理
